# dev_15: Crontab 문법 제조기 (Cron Generator & Parser)

## 개발 목표
- 사용자가 복잡하고 직관적이지 않은 Crontab(크론탭) 표현식을 시각적인 UI를 통해 쉽게 만들고(Generator), 거꾸로 크론 표현식을 입력하면 직관적인 자연어 한국어로 해석(Parser)해 주는 다기능 개발자 도구를 구현합니다.
- 전체 서비스의 표준 화면 규격(`maxWidth: 1600`, 일체화된 타이틀과 여백, 헤더 바 등)을 완벽하게 맞춘 프리미엄 MUI v6 UI/UX 구조를 적용합니다.

---

## 주요 구현 사항

### 1. 직관적이고 미려한 UI/UX 구조 (`src/pages/CronTool.tsx`)
- **표준 레이아웃 적용**: 최외곽 컨테이너 `maxWidth: 1600`과 여백 설정 및 좌우 분할 구조 (`Grid size={{ xs: 12, lg: 7.5 }}`와 `Grid size={{ xs: 12, lg: 4.5 }}`)를 완벽히 통정하여 일체감을 완성했습니다.
- **빠른 프리셋 기능**: 자주 쓰이는 대표적인 크론 표현식 패턴(매 5분마다, 매시간 정각, 매일 자정, 매일 아침 9시, 매주 평일 아침 9시, 매월 1일 자정)을 상단에 칩 형태로 배치하여 원클릭으로 쉽게 불러오고 조작할 수 있도록 설계했습니다.
- **탭 형태 기반의 비주얼 제네레이터**:
  - `매 분` (Minutes): 슬라이더 또는 단순 선택을 통해 손쉽게 분 주기를 설정합니다.
  - `매 시간` (Hourly): 매 시간 주기 실행 또는 특정 시작 시각 지정 등을 세부 설정합니다.
  - `매 일` (Daily): 일 단위 실행 또는 특정 시간(시/분)에 정기 실행을 구성합니다.
  - `매 주` (Weekly): 실행할 요일을 다중 선택 칩을 통해 클릭하고 구체적인 시각을 정의합니다.
  - `매 월` (Monthly): 실행할 월(1~12월) 다중 선택 칩과 특정 일 및 시각을 정교하게 조합합니다.
  - `고급 커스텀` (Custom): 5개 크론 필드(분, 시, 일, 월, 요일)의 인풋 필드를 직접 텍스트로 타이핑하여 특수 문자(와일드카드 `*`, `,`, `-`, `/` 등)를 자유롭게 실험해 볼 수 있도록 설계했습니다.

### 2. 강력한 크론 엔진 및 한국어 번역기 (Cron Engine & Translation)
- **한글 자연어 번역**: 난해한 크론 표현식을 읽고 한국어로 가장 직관적이고 세련되게 의미를 이해할 수 있도록 한글 번역 코드를 자체 구현했습니다.
  - 예: `*/5 * * * *` -> "매 5분마다"
  - 예: `0 9 * * 1-5` -> "매주 월요일부터 금요일까지 오전 9시 00분에 실행"
- **실시간 문법 검사 (Validation)**: 각 필드 범위(0~59분, 0~23시, 1~31일, 1~12월, 0~6요일) 및 특수 문자 조합 문법이 어긋난 경우, 친절하게 오류 메시지를 띄워 실시간 수정이 가능하게 합니다.

### 3. 다음 5회 실행 예정 시각 예측 (Execution Predictor Timeline)
- **실시간 시뮬레이터**: 클라이언트의 현재 PC 시각을 기준으로 입력된 크론 식을 매 분 시뮬레이션하여, 향후 실제로 스케줄러가 트리거될 **미래의 5회 실행 타임스탬프**를 예측 및 계산합니다.
- **타임라인 피드백 UI**: 계산된 일시를 가독성 높은 리스트 카드와 마이크로 호버 효과가 적용된 타임라인으로 렌더링하여 실시간 정밀성을 향상시켰습니다.

### 4. 컴파일 환경 최적화
- MUI v6 (Grid v2) 컴파일 기준에 부합하도록 기존 `Grid item` 속성을 최신 `Grid size`로 완전히 수정하였습니다.
- MUI v6 표준에 부합하게 `TextField`의 `InputProps` 속성을 `slotProps.input`으로 교정하여 타입 안정성을 확보했습니다.
- 사용되지 않는 React Default Import 및 미사용 MUI 컴포넌트(`Button`, `Checkbox`), 미사용 변수(`relative`)를 제거하여, 빌드 명령(`npm run build`)을 에러 및 경고 없이 성공 완료했습니다.
