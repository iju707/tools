# dev_12: 양방향 Base64 이미지 변환 기능 구현

## 개발 목표
- 기존의 단방향 이미지-Base64 변환 기능을 개선하여, Base64 텍스트 데이터를 입력하면 반대로 원래 이미지로 복원하여 화면에 렌더링해 주는 **양방향 변환 기능**을 구현합니다.
- 사용자가 복사해 둔 다양한 형태의 Base64 데이터(Data URI 포함 혹은 Raw Base64 형태)에 모두 유연하게 대응할 수 있도록 처리합니다.

## 주요 기능 (Features)

### 1. 이미지 파일 업로드 시 Base64 인코딩 (단방향)
- 기존과 동일하게 Drag & Drop 또는 파일 업로드를 통해 이미지 파일을 읽어 즉시 `Data URI`, `HTML img Tag`, `Raw Base64` 문자열로 변환합니다.

### 2. Base64 텍스트 입력 시 이미지 디코딩 및 렌더링 (역방향)
- **자동 감지 필드**: 텍스트 필드에 Base64 형식의 문자열을 입력하거나 붙여넣으면 실시간으로 디코딩을 수행합니다.
- **포맷 유연성**: 
  - `data:image/png;base64,...` 와 같은 **Data URI 형태**를 완벽하게 인식합니다.
  - 헤더 정보가 없는 **Raw Base64 형태**가 입력된 경우, 브라우저가 이미지로 렌더링할 수 있도록 접두사(`data:image/png;base64,`)를 자동으로 안전하게 삽입해 줍니다.
- **실시간 미리보기**: 변환에 성공하면 우측 영역에 복원된 이미지를 깨짐 없이 실시간으로 시각화합니다.

## 구현 상세
- **대상 파일:** `src/pages/ImageEncoderTool.tsx`
- **텍스트 붙여넣기 및 디코딩 로직:**
  - 사용자가 입력한 문자열이 Base64 형식인지 감지하는 상태값(`base64Input`)을 추가하였습니다.
  - 정규식을 활용하여 사용자가 입력한 문자열에 이미지 MIME 타입(`data:image/...`) 정보가 포함되어 있는지 여부를 자동으로 판별합니다.
  - 유효하지 않은 Base64 포맷 입력 시, 미리보기 영역에 에러 메시지를 노출하지 않고 우아하게 처리하도록 예외 처리를 적용했습니다.
